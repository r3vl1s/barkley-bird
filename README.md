Barkley Bird
===========

Shane Barkley must escape the clutches of the evil Steve Harvey

A clone of a  MelonJS made "Flappy Bird" clone.

Play online at http://r3vl1s.github.io/barkley-bird/

Check out the original Clumsy Bird at http://ellisonleao.github.io/clumsy-bird/

## Running Locally

- Install [Node](http://nodejs.org/download/) and [Grunt](http://gruntjs.com/)
- Install the dependencies

```
npm install
```

Then just type on your shell:

```
grunt connect
```

Open your browser at `http://localhost:8001/`
